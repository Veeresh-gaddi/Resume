# Resume / Biodata Website

This repository contains a small static resume/biodata website with the
following files:

- `resume.html` - main resume page
- `biodata.html` - biodata page
- `style.css` - styling for both pages

To run locally, open `resume.html` or `biodata.html` in your browser.

Feel free to update the README with more details (author, license,
deployment instructions, etc.).
